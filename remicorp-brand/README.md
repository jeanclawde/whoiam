# remicorp brand

Create a memorable visual identity for **remicorp**.

## Current phase

Moodboard and inspiration system for a corporate-modernist brand: 1955–1980 corporate identity, annual reports, standards manuals, and a small 2026 translation layer.

## First deliverable

- `index.html` and `site/index.html` — navigable resource-bank/moodboard webpage with filters, source links, color direction, and synthesis notes.
- Live GitHub Pages preview: https://jeanclawde.github.io/whoiam/remicorp-brand/
- `research/sources.json` — original structured source list.
- `research/sources-expanded.json` — expanded first-run source bank, 100+ resources.
- `research/resources-expanded.md` — readable expanded source bank.
- `research/moodboard-notes.md` — working notes and design implications.

## Working positioning draft

**remicorp** should feel like an institution that knows it is partly fictional: serious systems, annual-report authority, standards-manual rigor, but with enough wit and abstraction to be memorable. Avoid pure retro cosplay and avoid generic AI/SaaS gloss.
